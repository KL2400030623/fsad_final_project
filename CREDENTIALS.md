## Data Isolation

- **Doctors:** Each doctor only sees appointments where they are listed as the doctor
- **Patients:** Each patient only sees their own appointments, prescriptions, and medical records
- **Pharmacists:** Can see all prescriptions to fulfill them
- **Admin:** Can view and manage all data across all roles

## Adding New Users

Use the "Create Account" feature on the login page to add new **Doctor** and **Pharmacist** staff members only. Patient and Admin accounts are not created from this form.
